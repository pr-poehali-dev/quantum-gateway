# Галерист — Портфолио художника

![Preview](https://cdn.poehali.dev/templates/artiste-ru/preview-ru.jpg)

Это готовый сайт для арт-галереи или художника, который хочет красиво представить свои работы в интернете. Шаблон выполнен в элегантном светлом стиле с изысканной типографикой, которая подчеркивает ценность искусства.

На сайте есть все необходимые разделы: яркая главная страница с приветствием, галерея избранных художников с подробными карточками (биография, известные работы, портреты), каталог произведений искусства, информация о различных направлениях и стилях живописи, а также раздел об арт-пространствах.

Посетители могут узнать больше о каждом художнике, просто кликнув на его карточку — откроется красивое всплывающее окно с детальной информацией. Также есть удобная форма обратной связи для тех, кто хочет приобрести работу или задать вопрос.

Шаблон идеально подойдет для частных галерей, художников, коллекционеров или арт-дилеров, которым нужен стильный и профессиональный сайт без лишних сложностей.
